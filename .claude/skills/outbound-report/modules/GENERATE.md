# Generate Module

Calculate and format outbound campaign metrics.

## Metric Definitions

| Metric | Formula | Benchmark |
|--------|---------|-----------|
| Open Rate | unique opens / emails delivered | 50%+ good, 30-50% ok, <30% problem |
| Reply Rate | unique replies / emails delivered | 3-5% good, 1-3% ok, <1% problem |
| Positive Reply Rate | positive replies / total replies | 40%+ good |
| Meeting Rate | meetings / emails delivered | 0.5-1% good |
| Cost per Meeting | total spend / meetings booked | Under $200 target |

## Performance Classification

| Status | Criteria |
|--------|----------|
| Performing | Reply rate > 3%, positive reply > 40% |
| Needs Attention | Reply rate 1-3%, or positive reply 20-40% |
| Underperforming | Reply rate < 1%, or positive reply < 20% |

## Report Template

For each campaign, produce:

1. **Volume section**: Sent, delivered, bounced
2. **Engagement section**: Opens, clicks, replies
3. **Outcome section**: Meetings, pipeline, deals
4. **Insights section**: Top subject lines, best-performing segments
5. **Recommendations**: What to test next week
